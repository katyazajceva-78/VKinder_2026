# VKinder

## Описание
VKinder — это бот для ВКонтакте, который помогает находить новых друзей и знакомых. 
Он использует API ВКонтакте для поиска пользователей по заданным критериям и отображает их информацию в чате.
Бот также сохраняет информацию о найденных пользователях в избранное.

## Запуск
1. Установить зависимости
2. Вставить токены в config.py
3. Запустить bot.py

## Команды
- поиск: Начинает поиск пользователей по заданным критериям (пол, город, возраст). Бот отправляет информацию о найденном пользователе, включая имя, фамилию и ссылку на профиль, а также три самые популярные фотографии.


## Хранение данных
Файл favorites.json хранит информацию о пользователях, добавленных в избранное. Структура данных выглядит следующим образом:
[
    {
        "user_id": 123456789,
        "first_name": "Анна",
        "last_name": "Смирнова",
        "profile_url": "https://vk.com/id123456789"
    },
    ...
]

### Поля:
- **`user_id`**: Уникальный идентификатор пользователя ВКонтакте.
- **`first_name`**: Имя пользователя.
- **`last_name`**: Фамилия пользователя.
- **`profile_url`**: Ссылка на профиль пользователя.
